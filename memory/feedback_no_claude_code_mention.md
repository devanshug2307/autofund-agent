Do not mention "Claude Code" or "Built with Claude Code" in any public-facing content — tweets, README files, Moltbook posts, submission descriptions, or any external communications.

**Why:** User explicitly requested this. They don't want the tooling branded in their project's public presence.

**How to apply:** When writing tweets, READMEs, social posts, or submission metadata, describe the tech stack without referencing Claude Code as the development tool. Say things like "Built with Claude Opus 4.6" or just describe the agent framework without mentioning the harness.
